# Bolt Web Messenger

## How to Run it
1. Install dependencies with yarn.

* `yarn`

2. Run server.js with node to handle the websocket connection.

* `node server.js`

3. Open index.html in as many browser tabs as you like ->

![Bolt Nickname Selection](/samples/img01.png)

4. Have fun!
